List of Merge Refactorings

Definition: Replace an enum used for sorting logic with a dedicated Comparator (or key abstraction) to improve flexibility and maintainability.


-        GetSnapshotsRequest.SortBy sortBy,
+        SnapshotSortKey sortBy,

Rationale: Using a Comparator (or a sorting key abstraction like SnapshotSortKey) instead of an enum allows for richer, more flexible sorting logic without bloating the enum with multiple cases. It decouples sorting rules from the enumeration, making the code easier to extend and align with object-oriented design principles.

Reference: Elasticsearch Commit Example
